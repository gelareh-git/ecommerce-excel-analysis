# E-commerce Excel Data Analysis Project

## Overview
This project demonstrates an end-to-end Excel-based data analysis workflow using a multi-table European fashion e-commerce dataset.

## Dataset
The dataset includes the following tables:
- Sales
- Sales Items
- Customers
- Products
- Campaigns
- Channels
- Stock

## Key Analysis Areas
- Sales performance analysis
- Customer segmentation and tenure analysis
- Campaign and channel performance
- Product and category performance
- Time-based revenue trends
- Data quality issue identification

## Deliverables
- 18 Pivot Tables
- KPI Sheet with executive-level metrics
- Interactive Excel Dashboard
- Cleaned and structured Excel data model

## Tools Used
- Microsoft Excel
  - Pivot Tables
  - Advanced formulas (XLOOKUP, SUMIFS, UNIQUE)
  - Slicers
  - Dashboard design
